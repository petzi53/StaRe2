In-Text Terms

There are a few ways to customise the glossary term display.

  • Look up a term from the glossary file with glossary("alpha"): alpha

  • Display a different value for the term with glossary("alpha", "$\\alpha$"): \(\alpha\)

  • Use an inline definition instead of the glossary file with glossary("beta", def = "The second letter of the Greek alphabet"): betaThe second letter of the Greek alphabet

  • Just show the term (no hover) with glossary("effect size", show = "term"): effect size

  • Just show the definition with glossary("p-value", show = "def"):
